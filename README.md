# 📋 To-Do List Application

## 📝 Description
This is a simple Python-based To-Do List Application that helps users manage their daily tasks using a menu-driven interface. Users can add, view, delete tasks, and also track progress using a simple progress bar.

---

## ✨ Features
- ➕ Add one or multiple tasks  
- 📋 View current task list  
- ❌ Delete completed tasks  
- 📊 Display progress bar  
- 🎯 Simple and user-friendly menu interface  

---

## ⚙️ Requirements
- 🐍 Python 3.x  

---

## 📦 Modules Used
- ⏱️ `time` module (used for delay while deleting tasks)

---

## 🚀 How to Run
1. Open the program in any Python IDE (VS Code, PyCharm, IDLE, etc.)
2. ▶️ Run the Python file
3. 👉 Enter `y` to start the application
4. 📌 Choose an option from the menu:
   - 1 ➕ Add Task  
   - 2 ❌ Delete Task  
   - 3 📋 View Tasks  
   - 4 📊 Show Progress Bar  
5. 🚪 Enter `n` to exit the program  

---

## 🔄 Program Workflow
- 🟢 Program starts and shows menu  
- 👤 User selects an option  
- ⚙️ Action is performed  
- 🔁 Menu repeats until user exits  

---

## 📊 Output Includes
- 📋 Task list display  
- ✅ Success messages  
- ❌ Error messages  
- 📊 Progress bar  
- 👋 Exit message
